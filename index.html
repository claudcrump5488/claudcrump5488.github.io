STUDYING PROGRAMS

There are many avenues to explore at our Music Academy. Choose yours:

Guitar class program

1. The position of the hands. Basics of sound production. 2. Notation literacy, tablature. 3. Basic techniques of the game. 4. Technical excellence: exercises, scales, etudes. 5. Study of the classical and popular repertoire.

"Electric Guitar" class program

1. The position of the hands. Basics of sound production. 2. Notation literacy, tablature. 3. Basic techniques of the game. 4. Technical excellence: exercises, scales, etudes. 5. Study of the classical and popular repertoire. 6. The game is "minus". 7. Working with effects.

Bass Guitar class program

1. The position of the hands. Basics of sound production. 2. Notation literacy, tablature. 3. Basic techniques of the game. 4. Technical excellence: exercises, scales, etudes. 5. Study of the classical and popular repertoire. 6. The game is "minus". 7. Working with effects.

Vocal class program

1. Statement of voice. 2. Breathing. 3. Solfegging. 4. Voice science. 5. Study of the repertoire. 6. Working with backing tracks.

Piano class program

1. The position of the hands. Basics of sound production. 2. Basics of musical literacy. 3. Technical improvement. 4. Study of the repertoire. 5. Accompaniment.

Percussion class program

1. The position of the hands. Basics of sound production on a snare drum and kit. 2. The basics of musical literacy. 3. Rhythmic exercises and etudes. 4. Studying the rhythms of the main musical styles (rock, blues, pop, jazz, latina, etc.) 5. The game is "minus".
